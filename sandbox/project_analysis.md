# 沙盒物理模拟项目分析

## 项目概述
这是一个基于C++的沙盒物理模拟项目，模拟水、火种、沙子、木头等元素的物理行为和交互规则。

## 元素与规则

### 元素
- 水
- 火种
- 沙子
- 木头
- 燃烧的木头
- 空地

### 规则
1. **沙子规则**：如果下方是空的，沙子就掉下去
2. **水规则**：如果水旁边有空位，水就往旁边流
3. **木头规则**：木头放置的地方为非空地
4. **火种规则**：火种碰到木头后才会燃烧，变成燃烧的木头
5. **燃烧规则**：
   - 火种碰到木头，木头在3秒后变成燃烧的木头
   - 如果木头附近的有燃烧的木头，该块木头也将在5秒后变为燃烧的木头
6. **灭火规则**：水可以浇灭燃烧的木头，浇灭后燃烧的木头将消失，上面的沙子也会一起坠落

- **编程语言**：C++11/14/17
- **图形库**：SFML 或 SDL2
- **构建系统**：CMake
- **开发环境**：Visual Studio 或 VS Code

- **中等**：项目逻辑清晰，规则明确，适合C++实现
- **图形界面**：需要基础的2D渲染能力
- **物理模拟**：需要实现简单的物理规则和状态管理
