# Sandbox 沙盒物理模拟项目

## 项目简介
这是一个基于C++和SFML的沙盒物理模拟项目，模拟水、火种、沙子、木头等元素的物理行为和交互规则。

## 功能特点
- **元素交互**：模拟沙子下落、水流、燃烧等物理现象
- **用户交互**：使用鼠标左键放置元素，键盘按键切换元素类型
- **按键提示**：在屏幕上显示当前选择的元素和对应的按键
- **退出功能**：点击退出按钮或按ESC键退出程序
- **重力系统**：实现重力对沙子和水的影响

## 操作说明
- **放置元素**：使用鼠标左键在网格上点击
- **切换元素**：
  - A：沙子
  - S：水
  - D：木头
  - F：火种
  - E：空地
- **退出程序**：点击右上角的红色退出按钮或按ESC键

## 技术栈
- **编程语言**：C++17
- **图形库**：SFML 2.5.1
- **构建系统**：CMake 3.16+

## 构建和运行
1. 确保已安装SFML 2.5.1和CMake 3.16+
2. 进入build目录：`cd build`
3. 运行CMake配置：`cmake ..`
4. 构建项目：`cmake --build .`
5. 运行可执行文件：`SandboxSimulation.exe`

## 元素行为
- **沙子**：受重力影响，会向下和斜下方移动
- **水**：受重力影响，会向下流动，然后水平流动，可浇灭燃烧的木头
- **木头**：静态元素，可被火种点燃
- **火种**：会点燃周围的木头
- **燃烧的木头**：会在3秒后变为空地，会点燃周围的木头
