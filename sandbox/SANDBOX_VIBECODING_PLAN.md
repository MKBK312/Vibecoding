# Sandbox 沙盒物理模拟项目 - Vibecoding 计划文档

## 1. 项目概述

### 1.1 项目名称
Sandbox - 沙盒物理模拟项目

### 1.2 项目简介
这是一个基于纯前端技术的沙盒物理模拟项目，模拟水、火种、沙子、木头、泥巴等元素的物理行为和交互规则。项目使用HTML5 Canvas、CSS和JavaScript实现图形界面，展示了前端开发能力、物理模拟算法和用户交互设计，适合作为面试展示项目。

### 1.3 项目目标
- 实现一个功能完整的沙盒物理模拟系统
- 展示前端开发能力（HTML5、CSS、JavaScript）
- 展示物理模拟和状态管理算法
- 展示图形编程和用户交互能力
- 提供一个响应式和美观的界面设计

### 1.4 项目特点
- **物理模拟**：实现沙子下落、水流、燃烧、泥巴形成等物理现象
- **元素交互**：实现元素间的复杂交互逻辑（沙子+水=泥巴）
- **可视化**：提供美观的图形界面（响应式设计）
- **高性能**：优化的Canvas渲染和更新算法
- **跨平台**：支持所有现代浏览器

## 5. 核心功能实现

### 5.1 元素行为实现

#### 5.1.1 沙子（Sand）
- **颜色**：黄色 rgb(255, 255, 0)
- **行为**：
  - 检测周围是否有水
  - 如果有水，变成泥巴
  - 水不消失

#### 5.1.2 水（Water）
- **颜色**：蓝色 rgb(0, 0, 255)
- **行为**：
  - 向下流动（优先）
  - 水平流动（次选）
  - 只移动到空单元格
  - 不能部署在木头或燃烧的木头上

#### 5.1.3 木头（Wood）
- **颜色**：棕色 rgb(139, 69, 19)
- **行为**：
  - 静态元素，不移动
  - 只能被火种点燃
  - 其他元素不能部署在木头上

#### 5.1.4 火种（Fire）
- **颜色**：红色 rgb(255, 0, 0)
- **行为**：
  - 可以部署在木头上
  - 点燃木头使其变成燃烧的木头

#### 5.1.5 燃烧的木头（BurningWood）
- **颜色**：橙色 rgb(255, 165, 0)
- **行为**：
  - 燃烧3秒后消失
  - 可以点燃周围的木头
  - 其他元素不能部署在燃烧的木头上

#### 5.1.6 泥巴（Mud）
- **颜色**：深棕色 rgb(101, 67, 33)
- **行为**：
  - 沙子接触水时生成
  - 下沉（与水交换位置）
  - 水不消失，与泥巴共存

### 5.2 物理引擎实现

#### 5.2.1 网格更新策略
```javascript
class PhysicsEngine {
    update() {
        // 第一步：沙子接触水变成泥巴
        this.sandToMud();

        // 第二步：泥巴下沉（与水交换位置）
        this.mudFalling();

        // 第三步：水流动（只移动到空单元格）
        this.waterFlow();
    }
}
```

#### 5.2.2 性能优化策略
1. **批量更新**：减少DOM操作
2. **双缓冲**：使用两个Canvas图层
3. **空间分区**：只更新变化的区域
4. **requestAnimationFrame**：平滑动画

### 5.3 用户交互实现

#### 5.3.1 鼠标交互

#### 5.3.3 部署规则

2. **燃烧的木头**：所有元素都不能部署
3. **其他单元格**：所有元素都可以部署

## 7. 性能优化

### 7.1 Canvas渲染优化
- 使用离屏Canvas（双缓冲）
- 批量绘制元素
- 避免频繁的clearRect
- 使用ImageData直接操作像素

### 7.2 计算优化
- 使用TypedArray（Uint8Array）
- 避免创建新对象
- 使用位运算
- 缓存计算结果

### 7.3 内存管理
- 避免全局变量污染
- 及时清理事件监听器
- 使用对象池模式
- 避免内存泄漏

## 10. 风险评估

### 10.1 潜在风险
1. **性能问题**：大规模网格可能导致Canvas性能下降
2. **兼容性问题**：不同浏览器可能存在兼容性问题
3. **移动端支持**：移动设备上的交互体验可能不佳

### 10.2 风险缓解策略
1. **性能优化**：使用双缓冲和批量渲染
2. **兼容性处理**：使用标准API，避免非标准特性
3. **响应式设计**：优化移动端布局和交互
